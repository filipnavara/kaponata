Add a developer identity

A developer identity is a certificate combined with its private key. It is usually a file with the .p12 or .pfx extension. Select the developer identity you want to upload, and the password used to protect the developer identity.

You can create a Developer Identity by exporting it from Xcode. A developer profile has a .p12 or .pfx extension.

You cannot upload files with the .cer extension because they do not include the private key required to sign iOS applications.
